## This repository — AI Digital Twin portfolio

Local-first developer portfolio with an **AI Digital Twin**, built with **Next.js 16**, **React 19**, **TypeScript**, and **Tailwind CSS 4**.

Canonical profile data: `src/lib/constants.ts` · Digital Twin grounding: `src/lib/digital-twin-knowledge.ts` · CV materials: `doc/assets/` and `public/cv/`

### Quick start

```bash
cd Profile/Kashif-profile-ai-main
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### Configuration

Copy `.env.example` → `.env` as needed.

- `NEXT_PUBLIC_APP_URL=http://localhost:3000`
- Sanity (`NEXT_PUBLIC_SANITY_PROJECT_ID`) optional — `/blogs` and `/studio` work empty until configured
- Force OpenRouter: `NEXT_PUBLIC_DIGITAL_TWIN_FORCE_PROVIDER=openrouter`
